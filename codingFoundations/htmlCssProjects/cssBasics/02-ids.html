CSS IDs

An ID is a UNIQUE identifier

ID notes

  • In Css an id starts with an octothrope (pound, hashtag, number, etc.) - #
  • Example used for ids: Targeting a specific header or an about sections
  • Each ID should only be used once per page - they are unique!
    This is important in JavaScript later on as well.
  • Use for one off styling.
