Position

The position property specifies the type of positions method used for an elemnt.

The four positions values are:

  • fixed:this will pin something to a spot and it will stay there as the user scrolls. (ex. nav bars) Fixed positions takes elements out of the normal document flow
  • static:this is the default setting. It is not affected by top, bottom, left, and right properties. It is always poositioned according to the normal flow of a page
  • relative:does not take the element out of hte normal document lfow. Still appears withing its space/div/container, but allows it to be offset a bit from the top, bottom, left, and right. These properties will move it away from its normal position. Other content will not be adjusted to fit into and gap lefy by the element
  • absolutetakes element out of the normal document flow. If a parent elemnt position is set to relativem and the child element is set to absolute the child elements new (0,0) coordinate will be the upper left hand corner of the parent elemtn instead of the whole page.

    • The z-index:propert allows you to override the normal stacking order of the page. The default z-index is 0, and you can set that value to be positive or negative. An element with a greater z-inxes will stack in front of an element with the lower z-index. If the two elements have the same z-index, or don't have one assigned, the element positioned last in the HTML will show on top. You must have a position assigned in order to assign a z-index.
