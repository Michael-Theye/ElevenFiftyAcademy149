Width and Height Notes

The width and heigh properties are used to set the width and heigh of an element.

The defualt setting for width and height is auto, which means the browser the width and height.

Other settings for width and height are:

  • Specified lenght values such as px & em
  • percentage of the containing element (ex 50%, 100% etc.)
