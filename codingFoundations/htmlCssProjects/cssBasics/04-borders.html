Borders

The border property allows you to specify the style, width, and color of an element's border.

Border Style

The border-style property can tak eup to four values (top border, right border, bottom border, and left border). The following border-style values are alowed:

  • dotted:dotted border
  • dashed:dashed border
  • solid:solid border
  • doubledouble border
  • groove3D groove border,depends on the border-color value
  • ridge3D ridge border, depends on the border-color value
  • inset:3d inset border, depends on the border-color value
  • none:no border
  • hidden:hidden border
  • mixed:you can set a different border style for the top, bottom, right, and left

Border Width

VERY IMPORTANT: border-width will not work unlesss you have border-style set!

Border-width can bbe set to a specific size (px, em, etc) or you can use one of the following pre-set settings:thin, meduim, thick. The bnorder-width property can also take up four values (top, right, bottom, left)

Border-color can be set by the following ways:
